What "recommended" means:
- Recommended = the single operating point selected for a vessel configuration from the MATLAB sweep. The code first prefers fully feasible points, then current-safe fallback points, then the lowest-penalty point overall.

Why the same star appears on every constraint/performance plot:
- The star marks the same comparison-based recommended operating point on every constraint/performance panel because all panels share the same voltage-flow plane at one pitch; only the plotted response variable changes.
- In this case the starred point is 10.0 mm, 16.3 V, 40.0 L/min, so every panel shows the same x-y location while the z-value changes by metric.
