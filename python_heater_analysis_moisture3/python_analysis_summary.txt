What "recommended" means:
- Recommended = the single operating point selected for a vessel configuration from the MATLAB sweep by the shared lexicographic criteria used throughout the study. Earlier criteria dominate later criteria completely.

Why the same star appears on every constraint/performance plot:
- The standalone Heat Transfer Study plots do not place stars on the contour panels. Recommended operating points are reported in the text summary.
- In this case the starred point is 6.0 mm, 72.3 V, 400.0 L/min, so every panel shows the same x-y location while the z-value changes by metric.
