Selection process:
  Heating recommendations now use constrained NSGA-II on the existing evaluated design space.
  MATLAB still provides the physics and the full discrete candidate table; the Python layer
  then runs NSGA-II with integer decision variables on pitch, voltage, and total-flow indices.
  NSGA-II settings                      : population 100, generations 500, seed 7
  Step 1: compute the inequality-constraint residuals listed below for every candidate point.
  Step 2: run NSGA-II with those residuals as hard constraints and the objective set listed below.
  Step 3: keep only fully feasible points on the resulting Pareto front.
  Step 4: report one design point from that feasible Pareto set using the configured
          report-priority order. If the feasible Pareto set is empty, report no recommended
          design point and retain only the best-available reference point for diagnostics.
  No weighted penalty score is used to collapse the objectives before the Pareto step.
  Criteria settings:
    spread limit                       : 5.0 C
    minimum bottom temperature         : 22.0 C
    minimum top temperature            : 22.0 C
    maximum bottom temperature         : 26.5 C
    maximum top temperature            : 26.5 C

     1. current excess above the supply limit (A)
     2. wire-temperature excess above the limit (C)
     3. outlet-air excess above the limit (C)

Optimization process:
  The optimization table is now the feasible NSGA-II Pareto set for the active configuration,
  ordered by the configured reported-design priority rather than by a single blended score.
  Only points that satisfy all heating constraints and meet the full heating load appear here.
  The reported design point is the first entry in that ordered feasible Pareto list.
  Interpretation for a real system:
    1. Use NSGA-II only to generate the feasible Pareto set; do not collapse the objectives into a weighted penalty score before the Pareto step.
    2. Keep true equipment and safety limits as hard constraints: current, wire temperature, outlet-air temperature, pressure drop, water loss, hole velocity, and full-load heating duty.
    3. Use report_priority_order only after the feasible Pareto set has been identified; this picks one reported design point without violating constraints.
    4. Adjust spread_limit_C, min_bottom_temp_C, min_top_temp_C, max_bottom_temp_C, max_top_temp_C, and the report-priority ordering to reflect the real system rather than tuning arbitrary penalty multipliers.

Perforation and moisture process:
  Per-hole release and moisture loss are evaluated explicitly for the aeration tubes.
  Source map                           : Frederickson et al. (2007) Table 3 particle-size data,
                                         Churchill-Bernstein hole convection, Lewis relation,
                                         and IAPWS saturation/latent-heat relations
  Perforation geometry                 : 100 holes/tube, 5.0 mm diameter, area per hole 0.00001963 m^2
  Tube release fraction                : 95.0% of each tube flow is released through the perforations
  Per-hole flow relation               : released tube flow / holes per tube
  Hole-velocity relation               : released tube flow / (holes per tube x area per hole)
  Evaporation relation                 : m_dot = min(m_diffusion, m_capacity, m_heat)
  Diffusion limit                      : m_diffusion = h_m A (rho_v,sat(T_surface) - rho_v,bulk)
                                         the diffusion, carrying-capacity, and heat-limited
                                         evaporation rates are all evaluated in kg/s
  Heat-mass analogy                    : h_m = h / (rho c_p Le^(2/3)); h comes from the hole-scale convection correlation
  Sensible-to-latent cap               : m_heat = q_available / h_fg
  Moisture-state inputs                : RH 80%, surface temp 20.0 C, wetted area 612.297 m^2, Lewis factor 1.00
  Wetted-area basis                    : particle-surface estimate from Frederickson et al. (2007), Table 3

  Representative diameters             : > 40 mm 60.0 mm, 20-40 mm 30.0 mm, 10-20 mm 15.0 mm, < 10 mm 5.0 mm
