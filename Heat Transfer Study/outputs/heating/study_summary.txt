Selection process:
  One consistent lexicographic rule is used everywhere in this study:
    1. Vessel-comparison recommended point selection.
    2. Top candidate ranking for the active configuration.
    3. Wire-diameter hard-safe selected point at each sampled diameter.
    4. Optimization-ranked point table.
  Step 1: compute the hard-constraint residuals listed below for every candidate point.
  Step 2: define a hard-safe point as one with zero residual for current, wire temperature,
          outlet-air temperature, pressure drop, water loss, hole velocity, bottom temperature, top temperature,
          and bottom-top spread, plus positive net heat-to-bed output.
  Step 3: if one or more hard-safe points exist, rank only those points lexicographically
          with the ordered criteria below. Inside that hard-safe set, the clean physical
          rule is: maximize the colder-node bed temperature, then minimize power, then
          minimize bottom-top spread; mean bed temperature and heat shortfall are only
          late tie-breakers.
  Step 4: if no hard-safe point exists, report no recommended design point and fall back to
          the lexicographically best available point only as a plot/reference point.
  No blended penalty score is used anywhere in the selection logic.
  Criteria settings:
    spread limit                       : 5.0 C
    minimum bottom temperature         : 22.0 C
    minimum top temperature            : 22.0 C
  Hard-constraint residuals            : current, wire temperature, outlet-air temperature,
                                         pressure drop, water loss, hole-velocity excess, bottom-temperature shortfall,
                                         top-temperature shortfall, and spread excess
  Priority order:
     1. current excess above the supply limit (A)
     2. wire-temperature excess above the limit (C)
     3. outlet-air excess above the limit (C)
     4. pressure-drop excess above the limit (Pa)
     5. bottom-temperature shortfall below 22.0 C
     6. top-temperature shortfall below 22.0 C
     7. water-loss excess above 10.0 kg/24 h
     8. temperature-spread excess above 5.0 C
     9. negative colder-node temperature, so warmer colder-node values rank earlier
    10. total electrical power (W)
    11. raw bottom-top temperature spread (C)
    12. negative mean bed temperature, so warmer overall bed values rank earlier
    13. heat shortfall, expressed as max(0, dutyNeeded - 1)
  Wire-diameter study note:
    The wire plot now distinguishes hard-safe selected points from best-available
    reference points. If no hard-safe point exists at a sampled diameter, the solid
    selected curve is left blank there rather than dropping onto a last-resort point.

Optimization process:
  The optimization table is not a separate penalty-score solver.
  It is the same lexicographic ranking applied to the active-configuration design-point list.
  Hard-safe points rank ahead of violating points. If no hard-safe points exist, the table
  reports the least-violating best-available points and labels the active failure mode.
  Clean objective inside the hard-safe set:
    1. maximize min(T_bottom, T_top)
    2. minimize total electrical power
    3. minimize |T_bottom - T_top|
    4. use mean bed temperature and heat shortfall only as late tie-breakers
  Interpretation for a real system:
    1. Keep true equipment and safety limits as hard constraints: current, wire temperature, outlet-air temperature, and pressure drop.
    2. Treat bed-temperature minimums, water-loss limit, and maximum allowed bottom-top spread as hard process constraints, not soft penalties.
    3. After the hard constraints are satisfied, prefer the point with the warmer colder-node temperature, then the lower electrical power, then the smaller bottom-top spread.
    4. Adjust threshold values such as spread_limit_C, min_bottom_temp_C, and min_top_temp_C to reflect the real system rather than tuning arbitrary penalty multipliers.

  Recommended design point             : best lexicographic point among hard-safe candidates only
  Best-available reference point       : best lexicographic point in the full sweep when no hard-safe

  Clean ordering inside the hard-safe set
                                         : maximize min(T_bottom, T_top), then minimize power,

                                           ties with mean bed temperature and heat shortfall
  Failure categorization order         : current, wire temperature, outlet air, pressure drop,
